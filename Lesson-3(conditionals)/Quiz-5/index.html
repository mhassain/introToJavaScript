Lesson-3: Conditionals

Use the following variables in your solution:
  • balance - the account balance
  • isActive - if account is active
  • checkBalance - if you want to check balance

      To print out the account balance with decimal points (i.e. 325.00), use
        the .toFixed() method and pass it the number of decimal points you want
        to use. For example, balance.toFixed(2) returns 325.00.
    

If checkBalance equals true and isActive equals false, then Your
        account is no longer active. should be printed to the console.
